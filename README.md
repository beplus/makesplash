```bash
$ brew install makesplash
```

## Usage
Command-Line Interface
#### Help
```bash
$ ./makesplash --help
```
or 
```bash
$ ./makesplash -h
```

#### Make
for create directory with icons from file `Image.png`
```bash
$ ./makesplash --file Image.png
```
or 
```bash
$ ./makesplash -f Image.png
```

#### Version 
```bash
$ ./makesplash --version
```
or
```bash
$ ./makesplash --v
```
